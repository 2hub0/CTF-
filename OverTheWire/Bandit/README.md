# 心得体会
由于是第一次玩这种游戏，同时对于Linux操作系统不够了解，很多命令的使用都要通过查看帮助文档或在网上查相关的用法，尤其是在有关权限的一些关，卡壳了很久，还有自己编写bash脚本的时候因为空格的问题debug了很久，后面的关基本每题都要花上半个小时以上。虽然进展很缓慢，但效果还算不错，对很多指令都有了深刻的印象，对一些参数的用法也有了新的认识。未来有时间再从头通关一遍加深记忆。
